Служебная информация:
* Проект сконвертирован в git с использованием svn2git: https://github.com/nirvdrum/svn2git
** Команда: svn2git svn://serv1/KVV_S --rootistrunk --username guest
** Update: svn2git --rebase
** Initial push: git push --all ssh://gitolite@jpoint-server/kvv
** Add remote: git remote add gitolite ssh://gitolite@jpoint-server/kvv
** Update&push: svn2git --rebase && git push gitolite master
* Замечания:
** CRLF: сейчай все текстовые файлы проекта храняться с Unix-like форматом завершения строки (LF), вместо виндового (CRLF)
** Имена всех русских файлов теперь убиты из-за неправильной конвертации
