Замечания:
* CRLF: сейчай все текстовые файлы проекта храняться с Unix-like форматом завершения строки (LF), вместо виндового (CRLF)
* Имена всех русских файлов теперь убиты из-за неправильной конвертации
* СМСЭС в svn храниться в виде нескольких проектов с собственными ветками Trunk, Branches и Tags. Но я вытаскивал исходники целиком, без разбиения на подпроекты
* При получении обновлений в git отображается проценты, в svn - нет
* Если был разрыв коннекта при получении данных в git, то могут быть проблемы с работой, в svn - нет
